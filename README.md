# Arabic Typo Wallpaper

Projet Web Tool en p5 js pour le cours de creative coding .  
L'idée c'est de faire un mini web tool qui permet de générer un fond d'écran avec un prenom écrit en **arabe**.

## Le concept :
L'utilisateur met son prénom en en francais (typo Latine) et le webtool le traduit en arabe  
(ex souhil ->  سهيل)  
et le code crée un fond d'écran avec un **gros texte** + un **slash coloré** derriere.

L'utilisateur pourra donc changer :
- le texte arabe
- la couleur principal (accent)

## L'avancé du projet':
Dans cette premiere version du webtool, l'utilisateur peut écrire son prenom et le webtool le traduit en typo arabe et l'affiche en grand.


## Les langages utilisés :
- P5.JS
- HTML 
- CSS
